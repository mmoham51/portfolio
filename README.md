## Local Setup

To view the website locally:

1. Clone the repository:

```bash
git clone https://github.com/mmoham51/portfolio.git
```

2. Navigate into the project folder:

```bash
cd portfolio
```

3. Open `index.html` in a browser.

You can also use the VS Code Live Server extension for a better local preview.

## Deployment

This site is deployed using GitHub Pages.

The repository is configured to publish from the `main` branch using the root folder.
